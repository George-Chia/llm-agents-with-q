Solve a question-answering task using interleaved Thought, Action, and Observation steps. Thought involves reasoning about the current situation, and Action can be one of two types:
1. Select[], which selects candidate relations from the knowledge graph and returns a relation. Choose this action if the question cannot be answered based on the current observation.
2. Finish[answer], which returns the answer and ends the task. Choose this action if the question can be answered based on the current observation.
After each observation, provide the subsequent Thought and the following Action.
